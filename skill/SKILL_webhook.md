# 飞书交易命令

## 可用命令

| 命令 | 功能 | 示例 |
|------|------|------|
| /持仓 | 查询当前持仓 | /持仓 |
| /账户 | 账户摘要 | /账户 |
| /订单 | 活动订单 | /订单 |
| /成交 | 成交记录 | /成交 |
| /status | 监控状态 | /status |
| /start | 启动监控 | /start |
| /stop | 停止监控 | /stop |
| /refresh | 刷新数据 | /refresh |
| /log | 查看日志 | /log |
| /交易模式 | 切换到交易模式 | /交易模式 |
| /查询模式 | 切换到查询模式 | /查询模式 |
| /help | 显示帮助 | /help |

## 注意事项

- **飞书只支持查询功能**，不支持下单
- 查询类命令无需参数，直接发送即可
- 命令不区分大小写，支持中英文
