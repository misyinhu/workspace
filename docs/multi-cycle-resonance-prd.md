# 交易系统多周期共振分析 PRD

## 阶段一：项目背景

### 1.1 问题

**交易员当前痛点：**
- 在执行 IB 下单前，需要分析多个时间周期（如 1分钟、5分钟、15分钟、1小时等）的技术指标
- **现状**：需要手动在 TradingView 中来回切换不同周期，肉眼对比各周期的趋势一致性
- **问题**：切换耗时、容易遗漏关键信号、不同周期矛盾时难以快速决策

**数据源碎片化问题：**
- 不同品种来自不同数据源（国内期货、外盘、股票等）
- 缺乏统一的数据入口和标准化的数据格式
- 多源数据的时序对齐和一致性难以保证

### 1.2 目标

**核心目标：**
- 在 IB 下单界面直接集成多周期共振分析功能，让交易员无需切换窗口即可快速掌握各周期趋势一致性
- 提供量化的共振评分（0-100）和趋势矛盾识别，辅助交易决策
- 基于多周期共振/分歧情况生成简明的交易建议

**技术目标：**
- 利用已部署的 `quant-core` 统一数据基础设施，通过远程 API 获取多周期数据
- 支持多数据源（Tdx/IB/OKX/TradingView 等），通过配置化方式切换
- 保证数据实时性（延迟 < 3 秒）和数据完整性

**成功指标：**
- 交易员在多周期分析上的决策时间缩短 50% 以上
- 系统对多源数据的时序对齐误差控制在 1 秒以内
- 数据延迟超过 5 秒的情况 < 1%

### 1.3 背景分析与补充

**已有基础设施：**
- `quant-core` 已部署，提供统一的数据源接口（Tdx/IB/OKX/TradingView 等）
- 支持本地部署和远程 HTTP API 两种访问方式
- 已提供 `/api/history`、`/api/realtime` 等标准端点

**技术约束：**
- TradingView 数据源主要提供技术指标分析，不直接提供历史 OHLCV 数据
- TdxQuant 需要通达信软件在本地运行
- IB 数据源需要 IB Gateway 支持
- 多数据源切换需要通过 `source` 参数指定

**风险点：**
- 数据源延迟或断连时的降级策略
- 多周期数据时序对齐的准确性
- 不同数据源之间的数据格式差异
- 交易员过度依赖系统建议而忽视市场实际情况

## 阶段二：专业领域识别

**识别结果：** 本需求属于**金融交易系统**领域的**量化分析工具**开发，涉及**多时间周期技术分析**和**共振分析**。

**无需加载特定领域 skill**，采用通用产品需求文档编写流程。

**专业术语说明：**
- **多周期共振**：多个时间周期的技术指标同时发出相同方向的信号，增强趋势判断的可靠性
- **趋势一致性评分**：量化指标，表示各周期趋势方向的统一程度，范围 0-100
- **分歧系数**：量化指标，表示各周期趋势方向的矛盾程度，范围 0-100

## 阶段三：用户故事

### US-01：从品种库选择交易品种进行分析

- **角色**：作为 交易员
- **场景**：当我准备进行交易分析时，需要从系统中的品种库选择一个或多个品种
- **意图**：我希望能够方便地浏览和搜索品种库，选择我想分析的品种（如国内期货、外盘、股票等），支持选择单个品种或两个关联品种（如跨品种套利）
- **动机**：以便我开始针对特定品种进行多周期共振分析，不同品种可能来自不同数据源（国内期货、外盘、股票等），需要统一管理

### US-02：快速查看多周期趋势一致性

- **角色**：作为 交易员
- **场景**：当我选定品种后，准备执行 IB 下单前，需要快速了解当前品种在不同时间周期的趋势是否一致
- **意图**：我希望在下单页面一眼看到各周期（如1分钟、5分钟、15分钟、1小时）的趋势方向（上涨/下跌/震荡）以及它们的一致性评分
- **动机**：以便我能快速判断是否多个周期共振，提高决策效率，减少手动切换周期的时间

### US-03：识别多周期矛盾与分歧信号

- **角色**：作为 交易员
- **场景**：当我在下单页面查看多周期分析时，发现不同周期的趋势信号存在矛盾（如短周期看涨但长周期看跌）
- **意图**：我希望系统能明确标出哪些周期存在分歧，并提供分歧程度的量化指标
- **动机**：以便我能意识到潜在的风险，避免在周期矛盾时盲目下单，或者寻找更明确的共振机会

### US-04：获取基于多周期共振的交易建议

- **角色**：作为 交易员
- **场景**：当多周期分析显示明确的共振信号（多个周期趋势一致）或明显分歧时
- **意图**：我希望系统能基于当前的共振/分歧情况，给出简明的交易建议（如"建议买入 - 多周期共振看涨"、"建议观望 - 周期矛盾，等待更明确信号"等）
- **动机**：以便我能快速做出决策，特别是在高波动市场环境下节省时间，降低因周期切换导致的决策延迟

### US-05：自定义多周期分析的时间周期组合

- **角色**：作为 交易员
- **场景**：当我需要针对不同交易策略（如日内交易、波段交易）分析不同的周期组合时
- **意图**：我希望能够自定义显示哪些时间周期（如选择1分钟、5分钟、15分钟或切换到30分钟、1小时、4小时等），并保存为预设模板
- **动机**：以便我能根据不同的交易品种和策略灵活调整分析维度，让工具更贴合我的实际交易习惯

## 阶段六：界面与交互设计（概要）

### 6.1 页面结构

**整体布局：**
- 左侧边栏：品种选择区（可折叠）
- 顶部：市场状态总览条（共振分数、当前建议）
- 中部：多周期分析面板（各周期趋势卡片网格布局）
- 右侧：详细信息区（共振详情、矛盾分析、风险提示）
- 底部：快捷操作区（一键采纳、IB 下单按钮）

**区块划分：**
1. **品种选择区**：搜索框、品种列表、已选品种信息
2. **周期配置区**：周期选择器、预设模板、自定义设置
3. **多周期分析面板**：各周期趋势卡片（包含趋势方向、当前价格、涨跌幅）
4. **共振分析区**：共振分数仪表盘、共振等级标识、趋势分布统计
5. **矛盾识别区**：矛盾周期对列表、分歧系数、风险提示
6. **交易建议区**：当前建议卡片、建议详情、历史建议回顾
7. **快捷操作区**：一键采纳按钮、IB 下单按钮、设置按钮

**区块关系：**
- 品种选择 → 触发数据加载 → 更新多周期分析面板
- 周期配置变化 → 重新计算共振分数 → 更新共振分析区
- 各周期趋势变化 → 重新检测矛盾 → 更新矛盾识别区
- 共振分数变化 → 重新生成建议 → 更新交易建议区

**默认状态：**
- 首次进入页面：品种选择区展开，显示默认品种列表；多周期分析面板显示默认周期组合（1分钟、5分钟、15分钟、1小时）；其他区域显示引导提示或空状态
- 已选择品种：各区域显示该品种的实时分析数据
- 数据加载中：各区域显示加载动画或骨架屏

### 6.2 交互流程

**操作名称：选择品种进行分析**
- **触发**：交易员在品种选择区点击某个品种，或使用搜索框搜索并选择品种
- **执行中**：品种卡片显示加载动画；多周期分析面板显示数据加载中状态
- **成功**：品种卡片高亮显示为已选中状态；多周期分析面板显示该品种各周期的实时数据；共振分析区更新共振分数；如有数据异常，显示相应警告提示
- **失败**：品种卡片显示错误图标和提示；显示错误提示（如"数据加载失败，请重试"）；提供重试按钮
- **边界条件**：连续快速选择不同品种：防抖处理，只加载最后一次选择的品种数据；选择无权限品种：提示"您没有该品种的交易权限"；品种数据延迟超过10秒：提示"数据延迟严重，请谨慎决策"

**操作名称：自定义周期组合**
- **触发**：交易员点击"周期设置"按钮，打开周期配置面板
- **执行中**：周期配置面板从侧边滑入，显示当前已选周期和可选周期列表
- **成功**：勾选或取消勾选某个周期：多周期分析面板立即响应，添加或移除对应周期的卡片；选择预设模板（如"日内交易组合"）：自动勾选对应周期组合；点击"保存为预设"：弹出输入框，输入预设名称后保存；点击"恢复默认"：恢复系统默认的周期组合
- **失败**：选择的周期数少于2个：提示"至少需要选择2个周期"，阻止保存；选择的周期数超过8个：提示"最多只能选择8个周期"，阻止继续选择；保存预设时名称已存在：提示"该预设名称已存在，请更换名称"
- **边界条件**：无

**操作名称：查看共振详情**
- **触发**：交易员点击共振分数仪表盘或"查看详情"按钮
- **执行中**：展开共振详情面板，显示详细的共振分析数据
- **成功**：显示各周期趋势方向的详细对比；显示各周期之间的两两一致性矩阵；显示共振分数的历史变化曲线；显示当前共振等级的详细解释和参考案例；提供"导出报告"按钮，可将当前共振分析导出为PDF
- **失败**：数据加载失败：显示"数据加载失败"提示，提供"重试"按钮
- **边界条件**：无

**操作名称：采纳交易建议**
- **触发**：交易员点击"一键采纳建议"按钮
- **执行中**：显示确认对话框，展示建议详情和风险提示
- **成功**：点击"确认采纳"：自动跳转到 IB 下单确认页面；预填充下单表单：品种代码、买卖方向（根据建议）、数量（可编辑）、价格类型（默认市价）；显示建议来源和共振分数作为参考信息；提供"修改建议参数"按钮，可调整数量、价格类型等；提供"取消并返回"按钮，返回到多周期分析页面
- **失败**：IB 连接失败：提示"无法连接到 IB 交易系统，请检查网络或联系管理员"；品种无交易权限：提示"您没有该品种的交易权限"；当前时间不在交易时段：提示"当前不在该品种的交易时段内"
- **边界条件**：建议已过期（生成时间超过有效期）：提示"该建议已过期，请刷新查看最新分析"；市场状态发生重大变化（如共振分数变化超过20分）：提示"市场状态已发生变化，建议重新评估后再下单"
