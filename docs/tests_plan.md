# 测试用例设计文档

## 1. 核心功能测试

### 1.1 价差计算与信号生成

#### 1.1.1 正常场景
- **测试用例1**: 7日历史统计已知，当前价差明显低于历史最小值
  - 输入: 
    - asset1_price = [100, 101, 102, 103, 104, 105, 106]
    - asset2_price = [50, 51, 52, 53, 54, 55, 56]
    - multiplier1 = 2.0
    - ratio1 = 1
    - multiplier2 = 0.5
    - ratio2 = 2
    - threshold = 1000
  - 预期输出: signal = "MAX_SIGNAL"

- **测试用例2**: 价差回到历史区间内，无信号
  - 输入: 
    - asset1_price = [100, 101, 102, 103, 104, 105, 106]
    - asset2_price = [50, 51, 52, 53, 54, 55, 56]
    - multiplier1 = 2.0
    - ratio1 = 1
    - multiplier2 = 0.5
    - ratio2 = 2
    - threshold = 1000
  - 预期输出: signal = "NO_SIGNAL"

- **测试用例3**: 极端异常输入处理（如缺失价格、NaN）
  - 输入: 
    - asset1_price = [100, 101, None, 103, 104, 105, 106]
    - asset2_price = [50, 51, 52, 53, 54, 55, 56]
    - multiplier1 = 2.0
    - ratio1 = 1
    - multiplier2 = 0.5
    - ratio2 = 2
    - threshold = 1000
  - 预期输出: 抛出适当的自定义异常或返回稳定的 NO_SIGNAL

#### 1.1.2 边界场景
- **测试用例4**: 价差刚好等于阈值
  - 输入: 
    - asset1_price = [100, 101, 102, 103, 104, 105, 106]
    - asset2_price = [50, 51, 52, 53, 54, 55, 56]
    - multiplier1 = 2.0
    - ratio1 = 1
    - multiplier2 = 0.5
    - ratio2 = 2
    - threshold = 1000
  - 预期输出: signal = "MAX_SIGNAL"

- **测试用例5**: 价差刚好等于历史极值
  - 输入: 
    - asset1_price = [100, 101, 102, 103, 104, 105, 106]
    - asset2_price = [50, 51, 52, 53, 54, 55, 56]
    - multiplier1 = 2.0
    - ratio1 = 1
    - multiplier2 = 0.5
    - ratio2 = 2
    - threshold = 1000
  - 预期输出: signal = "NO_SIGNAL"

## 4. Webhook 桥接测试
