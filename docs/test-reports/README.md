# 测试报告目录

> 本目录存放归档测试报告。
> **模板**：[test-report-template.md](../../../../pmo/docs/test-report-template.md)

---

## 产出时点

| 触发条件 | 产出 |
|---------|------|
| 交付用户前 | 交付报告 |
| Sprint 结束 | Sprint 总结 |
| Issue 闭环 | 复测报告 |

**日常验证只更新 verified-facts.md，不产出报告。**

---

## 命名规范

```
test-report-[YYYY-MM-DD]-[类型].md
```

示例：
- `test-report-2026-05-09-delivery.md`（交付报告）
- `test-report-2026-05-09-sprint-1.md`（Sprint 总结）
- `test-report-2026-05-09-issue-1.md`（Issue 复测）
